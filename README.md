### [init.d-play]()


Configurable Init.d script for multiple Play 2 Framework apps

### installation  

( Tested on debian 7.0 wheezy Linux )

    # cd /etc/init.d
    # wget https://raw.github.com/julianromera/init.d-play/master/play
    # update-rc.d play defaults
